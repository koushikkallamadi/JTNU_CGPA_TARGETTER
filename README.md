# 🎯 JNTU CGPA Target Planner

A web-based application designed to help students calculate, analyze, and plan their target CGPA in a simple and structured way.

---

## 🚀 Live Demo

🌐 https://koushikkallamadi.github.io/JTNU_CGPA_TARGETTER/

---

## 📖 About The Project

The **JNTU CGPA Target Planner** helps students:

- Calculate current CGPA
- Plan future semester targets
- Understand required performance to reach a desired CGPA
- Manage academic goals effectively

This project was built to make academic planning easier and more interactive for students.
