## 📖 About The Project

The **JNTU CGPA Target Planner** is an all-in-one solution for academic goal setting. It empowers students to:

*   **Calculate Current CGPA:** Instantly find your overall CGPA based on previous semester SGPAs.
*   **Plan Future Targets:** Determine the required SGPA in upcoming semesters to reach a target CGPA.
*   **Predict Performance:** Understand how individual subject grades affect your overall standing.
*   **Official Formulas:** Uses official JNTU credit-based grading system (10-point scale).

This project was developed to bridge the gap between academic requirements and effective goal planning, making it easier for students to stay on track.

---

## ✨ Key Features

| Feature | Description |
| :--- | :--- |
| **🎯 CGPA Targeter** | Calculate required SGPA for future semesters to hit your goal. |
| **📊 SGPA Calculator** | Dynamic calculation based on individual subject credits and grades. |
| **📈 CGPA Calculator** | Aggregate your semester SGPAs into a single weighted CGPA. |
| **🔢 Percentage Converter** | Accurate conversion using the formula: `(CGPA - 0.5) * 10`. |
| **📱 Responsive UI** | Fully optimized for mobile, tablet, and desktop viewing. |
| **📘 Grading Guide** | Integrated reference for the JNTU 10-point grading system. |

## 🛠 Getting Started

Establishing the project locally is simple and requires no complex setup.

### Prerequisites
- A modern web browser (Chrome, Firefox, Edge, or Safari).

### How to Run Locally
1.  **Clone the Repository:**
    ```bash
    git clone https://github.com/koushikkallamadi/JTNU_CGPA_TARGETTER.git
    ```
2.  **Navigate to the project folder:**
    ```bash
    cd JTNU_CGPA_TARGETTER
    ```
3.  **Open the Project:**
    - Simply double-click on `index.html` to open it in your default browser.
    - *Alternatively*, if you are using VS Code, use the **Live Server** extension for a smoother development experience.

## 📖 How It Works

1.  **Select a Tool:** Choose from CGPA Targeter, SGPA Calculator, or Percentage Converter.
2.  **Input Data:** Enter your credits and grade points or previous semester SGPAs.
3.  **View Results:** The tool instantly calculates and displays your result using official JNTU formulas.
4.  **Plan Ahead:** Use the Targeter to see exactly what you need to score in the next semester.
